# Hangman (Виселица)

Консольная реализация игры "Виселица" на Java. Проект выполнен в рамках [Java-роадмапа Сергея Жукова](https://zhukovsd.github.io/java-backend-learning-course/projects/hangman/).

## Демо

```
================================
   ┌──────────┐
   │          │
   │         ( )
   │         \│/
   │          │
   │           
   │           
───┴───────────
 р_б_т_
================================
    YOU HAVE ALREADY ENTER:
 р, б, т,
================================
================================
=    ENTER YOUR LETTER HERE:   =
```

## Функциональность

- Случайное слово выбирается из словаря русских существительных
- При старте одна буква открывается автоматически
- После каждого хода отображается виселица в ASCII-арте (6 стадий)
- Показываются уже введённые буквы
- По завершении раунда можно начать новую игру или выйти

## Технологии

- Java 17
- Maven

## Запуск

```bash
git clone https://github.com/icefekDev/Hangman.git
cd Hangman
mvn compile exec:java -Dexec.mainClass="Launcher"
```

Или через IDE: запустить `Launcher.java`.

## Структура проекта

```
src/main/java/
├── Launcher.java          # Точка входа, главный игровой цикл
├── entity/
│   ├── Judge.java         # Логика проверки букв и ведения слова
│   ├── Player.java        # Состояние игрока (ошибки, введённые буквы)
│   ├── WordPicker.java    # Выбор случайного слова из словаря
│   ├── HangmanUI.java     # Весь консольный вывод и ASCII-арт
│   └── Resetable.java     # Интерфейс для сброса состояния
└── tool/
    ├── ConsoleHandler.java    # Ввод с консоли
    └── WordsExtracter.java    # Чтение словаря из файла

src/main/resources/
└── Nouns.txt              # Словарь русских существительных
```

## Правила игры

1. Введите `старт` чтобы начать игру, `выход` чтобы выйти
2. Угадывайте слово вводя по одной букве
3. Допускается не более **5 ошибок**
4. При победе или поражении можно начать новый раунд
